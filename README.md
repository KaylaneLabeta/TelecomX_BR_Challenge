# 📊 Análise de Evasão de Clientes - Telecom X

## Descrição do Projeto
Este projeto tem como objetivo analisar o **churn (evasão de clientes)** da Telecom X. A análise identifica padrões de comportamento que levam clientes a cancelar serviços e fornece insights para melhorar **retenção de clientes**.

O projeto inclui:
- Importação e limpeza de dados.
- Análise descritiva e distribuição de churn.
- Contagem de evasão por variáveis categóricas e numéricas.
- Análise de correlação entre variáveis numéricas (extra opcional).
